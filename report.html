🚀 프로젝트 보고서: QR Dropzone

제작자: 유선재 (명지대학교 기계공학과 4학년)

1. 기획 배경 (Motivation)

팀 프로젝트나 발표 현장에서 PC 내 파일을 타인의 스마트폰으로 전송해야 할 때, 카카오톡 로그인이나 클라우드 업로드 과정은 번거롭고 개인정보 유출 위험이 있습니다. QR Dropzone은 복잡한 절차 없이 드래그 앤 드롭 한 번으로 보안 전송 환경을 구축하기 위해 제작되었습니다.

2. 주요 기능 및 실행 화면

[그림 1] 상시 대기 중인 드롭존 UI
[그림 2] 파일 드롭 후 자동 생성된 QR
  • 초경량 드롭존: 바탕화면 어디든 이동 가능한 반투명 위젯 형태로, 즉각적인 파일 수신 대기.
  • 동적 QR 시스템: 드롭된 파일의 고유 경로와 현재 PC의 로컬 IP를 조합하여 전송용 QR 코드 실시간 렌더링.

3. 실현 방법 및 기술 스택

Python 3.12 PyQt5 (GUI Framework) Socket & Threading HTTP Server Library

기술적 특징: 멀티스레딩 기법을 도입하여 UI 스레드와 파일 호스팅 서버 스레드를 분리하였습니다. 이를 통해 대용량 파일 전송 중에도 프로그램이 멈추지 않고 실시간 상태를 유지합니다. 또한, HTTP 헤더 최적화를 통해 모바일 기기에서 스캔 시 별도의 확인 절차 없이 즉시 다운로드가 시작되도록 설계하였습니다.

[파일 드롭 확인] ➔ [로컬 소켓 서버 개방] ➔ https://ko.qr-code-generator.com/ ➔ [스마트폰 HTTP 요청 수신]

4. 특장점

  • 보안성(Privacy): 외부망을 거치지 않는 로컬 통신으로 데이터 유출 원천 차단.
  • 범용성(Universal): Android, iOS 등 OS에 관계없이 표준 브라우저만 있다면 파일 수신 가능.
  • 직관성(UX): '파일 선택-업로드-링크 생성'의 3단계 과정을 '드래그 앤 드롭' 1단계로 축소.

5. 향후 계획 (Future Work)

현재 버전은 동일 Wi-Fi 환경 내 전송을 지원하며, 향후 다중 파일 일괄 전송 기능 및 전송 만료 시간 설정 기능을 추가하여 완성도를 높일 계획입니다.
